# 预设系统使用说明

## 功能概述

预设系统允许用户保存和管理音频转换参数配置，快速应用常用的转换设置，提高工作效率。

## 主要功能

### 1. 内置预设（8个）

系统提供了8个常用的内置预设，涵盖不同使用场景：

- **高质量MP3** - 320kbps MP3，适合高品质音乐
- **标准MP3** - 192kbps MP3，日常使用的平衡选择
- **压缩MP3** - 128kbps MP3，节省空间
- **无损FLAC** - FLAC格式，完全无损
- **高质量WAV** - 48kHz WAV，专业音频
- **播客优化** - 96kbps 单声道，适合语音播客
- **语音录制** - 64kbps 单声道，语音记录
- **音乐制作** - 96kHz WAV，专业音乐制作

### 2. 自定义预设

用户可以保存自己的参数配置：

1. 在主界面调整好参数（格式、质量、声道、编码、采样率）
2. 点击顶部的"预设"按钮
3. 在弹窗中点击"保存为预设"
4. 输入预设名称并保存

### 3. 预设管理

- **加载预设** - 点击任意预设即可应用其参数
- **重命名** - 点击自定义预设旁的编辑图标
- **删除** - 点击自定义预设旁的删除图标
- **导出** - 将自定义预设导出为JSON文件
- **导入** - 从JSON文件导入预设

### 4. 预设信息显示

每个预设显示以下信息：
- 预设名称
- 输出格式
- 质量（kbps）
- 编码方式（VBR/CBR）
- 声道设置
- 创建时间（仅自定义预设）

## 使用场景

### 场景1：快速切换工作模式

如果你经常需要在不同场景下转换音频：
- 为播客创建"播客优化"预设
- 为音乐创建"高质量MP3"预设
- 为语音备忘录创建"语音录制"预设

### 场景2：团队协作

导出预设文件分享给团队成员，确保大家使用相同的转换参数。

### 场景3：批量处理

为不同类型的音频文件准备不同的预设，快速切换参数进行批量转换。

## 技术实现

- **存储方式**: localStorage（浏览器本地存储）
- **数据格式**: JSON
- **预设ID**: 自动生成唯一标识
- **时间戳**: 记录创建时间

## 注意事项

1. 内置预设不可删除或修改
2. 自定义预设存储在浏览器本地，清除浏览器数据会丢失
3. 建议定期导出重要的自定义预设作为备份
4. 导入预设时会自动生成新的ID，不会覆盖现有预设

## 快捷操作

- 点击预设名称 → 立即应用并关闭弹窗
- 双击预设 → 快速应用
- ESC键 → 关闭预设管理窗口
- Enter键 → 保存新预设（在输入框中）
